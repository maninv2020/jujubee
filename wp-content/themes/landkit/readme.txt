Landkit is the perfect theme for your next WordPress website.

== Description ==

Landkit is a Multipurpose Business WordPress Theme. It is designed with the latest design trends in mind. Landkit feels more modern, minimal and beautiful. It is built to make your life easier. Build a beautiful, modern WordPress website with Landkit. Develop anything.

For more information about Landkit please go to https://themeforest.net/user/madrasthemes/portfolio

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click on 'Upload Theme' button and upload the theme package that is downloaded from ThemeForest.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to https://docs.madrasthemes.com/landkit/ guides on how to customize this theme.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
